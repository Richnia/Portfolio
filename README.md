# Portfolio Personnel – Développeur Web

## À propos du projet

Ce portfolio personnel est un projet de présentation réalisé avec les technologies du web moderne : HTML5, CSS3 et JavaScript.

Il a pour objectif de mettre en valeur mon profil, mes compétences et mes réalisations dans le domaine du développement web.

## Fonctionnalités

- Page d’accueil de présentation
- Mon parcours
- Tableau des compétences 
- Mes certifications
-  Mes Projets
- Mes liens

## Technologies utilisées

- HTML5
- CSS3
- JavaScript

## Structure du projet

- index.html : structure principale du site
- style.css : design et mise en forme
- model.js : interactions et logique JavaScript
- images/ : ressources visuelles &  attestations
- documents/ : fichiers complémentaires (PDF)

## Objectif

Ce projet vise à démontrer mes compétences en développement web frontend et à constituer une vitrine professionnelle de mon profil en informatique.
