⋄ Jeux de Survie Python

Développé sur la calculatrice Numworks avec Python.

Conception 2D

Objectif du projet

L'objectif de ce projet était de concevoir un jeu de survie en Python afin de mettre en pratique les notions de programmation, d'algorithmique et de logique de développement. Le joueur doit prendre des décisions stratégiques pour survivre dans différents scénarios tout en gérant ses ressources et les événements rencontrés au cours de l'aventure.

⋄ Robot Humanoïde Paul

Projet actuellement en construction.

En cours

Objectif du projet

Paul est un projet de robot humanoïde intelligent conçu pour faciliter les interactions entre les humains et les technologies d'intelligence artificielle. L'objectif est de développer un robot capable de communiquer en français ainsi que dans ma langue maternelle grâce aux technologies de traitement du langage naturel. À terme, il intégrera des fonctionnalités telles que la reconnaissance faciale, la reconnaissance vocale et la synthèse vocale afin d'interagir de manière naturelle avec les utilisateurs.

🔧 Technologies utilisées

  • Intelligence Artificielle
  • LLM
  • NLP
  • Reconnaissance vocale
  • Python
  • Robotique
